Rich Content — каталог компонентов

Универсальная вёрстка для встраивания на сайты Bort. Префикс классов brt-rich. Стили изолированы в блоках <style id="...">. Для кастомизации под сайт редактируйте #rich-vars и #rich-theme.

#rich-vars Heading Subheading Text Stat Badge Grid (row/col) Hero Statement Split Media Accent Bar Cards Accent Modes Card Params Logo Card Split Stats Media Text Columns Card Accent Cards Feature Grid Info Cards Contents Summary

Блок стилей #rich-vars

CSS-переменные — единственное место для кастомизации темы под конкретный сайт. Меняйте цвета, шрифты, скругления, пропорции колонок.

<style id="rich-vars"> … --rich-color-accent, --rich-radius-md, --rich-split-left … </style>

Блок стилей #rich-base

Структурные стили: 12-колоночная сетка brt-rich-row / brt-rich-col-* (модель gutter как в Bootstrap 5: padding на колонках + negative margin на row). Брейкпоинты: sm ≥576px, md ≥768px, lg ≥992px. Утилиты: brt-rich-col-auto, brt-rich-col-grow. Модификаторы row: --align-center, --reverse, --gutter-0, --hero, --gap-2xs, --gap-xs, --gap-sm, --gap-md.

Блок стилей #rich-theme

Применение переменных к компонентам: типографика, карточки, баннеры.

STYLE: #rich-theme → .brt-rich__stat

Атом: значение + подпись. По умолчанию центрирование; модификатор --left для выравнивания влево (specs, filters).
