bort-global api

bort-global api

Bort-Global.API обеспечивает возможность автоматизированного получения данных от компании Bort Global. Данный функционал постоянно дорабатывается и количество ресурсов будет увеличиваться. Если у вас есть замечания или пожелания, то просим передать их нам.

Все данные передаются в кодировке UFT-8.

Компания Bort Global оставляет за собой право ограничить вам доступ к Bort-Global.API в случае если будут замечены попытки нелегального доступа к ресурсам (в том числе передача URL'ов третьим лицам или попытки изменить формат URL'ов).

ресурсы api

товарная база

Данный запрос позволяет получить товарную базу (наименование, код товара, ссылки на фотографии, характеристики, ...) в различных форматах (вы можете выбрать самостоятельно). Эти данные помогут вам добавлять (обновлять) все товары на ваш сайт или Интернет-магазин.

Формат:
URL:

Описание форматов: VendorYML, YML

РРЦ, цена отгрузки и наличие товаров

Запрос позволяет получать рекомендованные розничные цены (РРЦ) и доступность товара на складе. Формат также можно выбрать самостоятельно. Ресурс будет полезен всем владельцам Интернет-магазинов для автоматического обновления наличия товаров и розничных цен.

Формат:
URL:

В формате CSV передаются поля (по-порядку): код товара, наименование, РРЦ, наличие на складе, цена отгрузки. Все значения заключены в кавычки и разделены точкой с запятой.
