Rich Content — каталог компонентов

Универсальная вёрстка для встраивания на сайты Bort. Префикс классов brt-rich. Стили изолированы в блоках <style id="...">. Для кастомизации под сайт редактируйте #rich-vars и #rich-theme.

#rich-vars Heading Subheading Text Stat Badge Spec card Feature card Mini card Step Split ratios Split + cards Hero Banner Gallery Statement Kit contents

Блок стилей #rich-vars

CSS-переменные — единственное место для кастомизации темы под конкретный сайт. Меняйте цвета, шрифты, скругления, пропорции колонок.

<style id="rich-vars"> … --rich-color-accent, --rich-radius-md, --rich-split-left … </style>

Блок стилей #rich-base

Структурные стили: grid, flex, breakpoints, контейнер. Не изменяйте без необходимости.

Блок стилей #rich-theme

Применение переменных к компонентам: типографика, карточки, баннеры.

Атом. Модификаторы: --h1, --h2, --h3, --center. Используется в hero, split, banner, feature-card.

Text (текст)

STYLE: #rich-theme → .brt-rich__text

Атом. Модификаторы: --small, --inverse (для тёмных/акцентных фонов).

Stat (характеристика)

STYLE: #rich-theme → .brt-rich__stat

Атом: значение + подпись. Используется внутри spec-card.

.brt-rich__stat-value + .brt-rich__stat-label
560 Вт Мощность

Badge (плашка)

STYLE: #rich-theme → .brt-rich__badge

Атом. Chip с акцентной обводкой. Используется в hero-badges.

.brt-rich__badge
Мощный мотор 3 ступени

Spec card (карточка характеристики)

STYLE: #rich-theme → .brt-rich__spec-card

Молекула: иконка + stat. Собирается в grid --specs (3 колонки на desktop).

.brt-rich__spec-card

Feature card (карточка преимущества)

STYLE: #rich-theme → .brt-rich__feature-card

Молекула: иконка + inline bold title + текст. Сетка --features (2 колонки, gap 10px).

.brt-rich__feature-card

Засоры в трубах. Измельченные отходы перемалываются и легко смываются в канализацию потоком воды, не создавая засоры.
